# Shopping Cart

A kata to practice writing focused integration tests using [test containers](https://testcontainers.com/?language=nodejs).


### Hints
1. [Left Join in MariaDB](https://www.mariadbtutorial.com/mariadb-basics/mariadb-left-join/)

2. Remember that if a table contains a foreign key that references another table, it cannot be dropped. In this case, it is necessary to drop the other table first.
